📌 Descripción
En esta actividad desarrollarás la landing page de Loopstudios utilizando Astro y Tailwind CSS, con el objetivo de poner en práctica tus conocimientos en la creación de componentes, uso de estilos y diseño responsivo.

El reto consiste en recrear la landing page lo más parecido posible al diseño proporcionado. Este ejercicio te permitirá fortalecer tus habilidades en el manejo de componentes de Astro, así como la organización y estilo visual mediante Tailwind CSS.

Podrás apoyarte en el diseño de referencia (versión móvil y de escritorio), tomando tus propias decisiones sobre tamaños, márgenes, tipografía y espaciado.

🎯 Objetivos de aprendizaje

Practicar la creación de componentes reutilizables utilizando Astro.

Aplicar conceptos de diseño responsivo mediante Tailwind CSS.

Reforzar el uso de HTML semántico y estructura de contenido.

Aprender a replicar un diseño visual con atención al detalle.

Desarrollar la capacidad de organizar y documentar su proyecto en un repositorio de GitHub.

✅ Requerimientos del reto
Tu proyecto deberá incluir los siguientes elementos:

Página principal que contenga todos los componentes de la landing page según la maqueta proporcionada.

Componentes de Astro para cada sección de la landing page (ej. header, hero, sección de galerías, footer).

Estructura semántica en HTML para todo el contenido.

Estilos implementados con Tailwind CSS, cuidando la tipografía, márgenes y espaciado.

Diseño responsivo, visible correctamente en versión móvil y escritorio.

Interacciones mínimas, como hover states y toggle del menú móvil (opcional con JavaScript).

Uso del archivo style-guide.md como referencia para paleta de colores y fuentes.

El proyecto debe mostrarse correctamente en un navegador al abrir la página principal.

🧠 Recomendaciones
Antes de comenzar:

Analiza el diseño y planea la estructura de tus componentes.

Escribe primero la estructura base con HTML y componentes de Astro antes de aplicar estilos.

Aplica los estilos de arriba hacia abajo, asegurando que cada sección quede completa antes de continuar.

Ajusta las medidas de acuerdo a tu propio criterio (usa padding, margin y font-size adecuados).

Revisa la documentación oficial de Astro y Tailwind CSS para resolver dudas.
